Valkey is an astonishingly fast & open source data store.

It is well suited for key/value workloads, yet also features a wide variety of native data structures.

Key features

Super speedy

Valkey is an in memory data store, making it faster than most disk based solutions.

Many data structures

Valkey could be described as a "data structure server". It supports strings, hashes, lists, sets, streams, & many more.

Persistent

Valkey saves all writes to permanent storage to survive reboots and system failures.

Extensible

Valkey has a modular API, facilitating the creation of custom extensions in C, C++, or Rust.

Scalable

Valkey supports horizontal scalability with hash-based sharding. It can automatically re-partition when growing the cluster.

Highly Avaliable

Replication with automatic failover for both standalone and clustered deployments.
